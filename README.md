## О проекте

Простой Telegram бот для получения `file_id` любых вложений (фото, документы, видео, стикеры и т.д.).

## Быстрый старт

### Docker (рекомендуется)

```bash
# Запуск
docker-compose up -d

# Просмотр логов
docker-compose logs -f

# Остановка
docker-compose down
```

### Локально

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск
python bot.py
```

## Настройка

Создайте файл `.env`:
```
BOT_TOKEN=your_bot_token_here
```

## Использование

1. Отправьте `/start` боту
2. Пришлите любое вложение
3. Получите его `file_id`

## Поддерживаемые типы файлов

- Фото
- Документы
- Видео
- Аудио
- Голосовые сообщения
- Видео-заметки
- Анимации (GIF)
- Стикеры
